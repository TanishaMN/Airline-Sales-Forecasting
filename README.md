# Airline Passenger Forecasting 

## Project Overview
This project analyzes monthly airline passenger data to predict future traffic using Time Series Forecasting.

## Key Results
* **Stationarity:** Achieved stationarity using **Second Order Differencing** (ADF p-value: ~2.73e-29).
* **Model:** Built an **ARIMA(1, 2, 1)** model.
* **Tools:** Python, Pandas, Statsmodels.
